# Calculator
Simple Calculator App Written in Swift.

Sample Operation
9 , Enter , 9 Enter , Multiply ( 9 , multiply , 9 will not work) 

Input number before you can do the operation

1. Optionals
2. Closures
3. enum
4. switch
5. Dictionary
6. Tuples
7. Autolayout
8. Recursion (not really an iOS thing, but something you should know!)
